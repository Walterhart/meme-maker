# Meme Maker

Meme Maker is a web application that allows users to generate custom memes by adding their own text to popular meme templates.

## Features

- Fetches a list of popular meme templates from an API.
- Allows users to enter their own text for the top and bottom sections of the meme.
- Displays a randomly selected meme when the "Fetch an image" button is clicked.
- Renders the selected meme with the user's text overlaid on the image.
- Provides a navigation menu for easy access to the home page and about page.

## Technologies Used

- React: JavaScript library for building user interfaces.
- CSS Modules: CSS styling approach that scopes styles to individual components.
- Fetch API: JavaScript interface for making HTTP requests.

## Getting Started

Follow the instructions below to get a copy of the project up and running on your local machine.

### Prerequisites

- Node.js: Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/walterhart/meme-maker.git
   ```

2. Navigate to the project directory:

   ```
   cd meme-maker
   ```

3. Install the dependencies:

   ```
   npm install
   ```

4. Start the development server:

   ```
   npm start
   ```

5. Open your web browser and visit `http://localhost:3000` to view the application.
